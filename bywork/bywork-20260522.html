解决方案：

import openai

def translate_to_english(chinese_text, style="professional"):
    prompt = f"""
    将以下中文翻译成{style}风格的英文，适合亚马逊产品描述：
    {chinese_text}
    要求：
    1. 保留SEO关键词
    2. 符合欧美用户阅读习惯
    3. 适当加入情感词
    """
    response = openai.ChatCompletion.create(
        model="gpt-4.5",
        messages=[{"role": "user", "content": prompt}]
    )
    return response.choices[0].message.content

五、风险提示与合规建议

5.1 主要风险

  1. API稳定性风险： 官方API可能调整价格、政策或限制调用，建议使用中转服务降低风险
  2. 版权风险： AI生成内容可能涉及版权问题，建议加入人工审核，注明AI生成
  3. 合规风险： 涉及金融、医疗、法律等敏感领域，需要相关资质，建议从轻量化工具切入
  4. 平台政策风险： 电商平台、内容平台可能调整对AI生成内容的政策，需要及时关注并调整

5.2 合规建议

  1. 数据隐私： 用户上传的文档、个人信息要做好加密存储，不要泄露
  2. 内容审核： AI生成的内容要经过人工审核，避免出现违规内容
  3. 明确标注： 向用户明确说明哪些内容是AI生成的，避免误导
  4. 用户协议： 制定清晰的用户协议和隐私政策，明确双方权利义务
