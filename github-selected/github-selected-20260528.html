6. NornicDB

GitHub: nornic/nornicdb
语言: Go

一句话评价

Graph+Vector+时序三合一，RAG场景的新一代数据库选择。

核心技术

多模态索引融合：

// 统一查询接口
result, _ := db.Query(`
    MATCH (a:Person)-[:KNOWS]->(b:Person)
    WHERE a.name = 'Alice'
    RETURN b, 
           vector_similarity(b.embedding, $query_emb) as score
    ORDER BY score DESC
    LIMIT 10
`)

性能指标：

指标 数值
