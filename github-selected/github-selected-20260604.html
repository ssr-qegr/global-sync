Windows Secure Runtime — Agent 安全基础设施

微软将 Windows 定位为 Agent 安全运行时，三大支柱：

  1. Containment（隔离）：OS 级 Agent 沙箱，限制操作范围和权限时效
  2. Identity（身份）：Agent 成为一级安全主体，审计日志可区分人与 Agent
  3. MXC（可管理性）：企业级 Agent 策略管理，OpenClaw 已接入

据WindowsForum 分析，这是微软承认"自治软件需要不同的安全模型"的标志。
