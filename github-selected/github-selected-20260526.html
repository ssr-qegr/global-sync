📈 今日技术趋势

趋势1：代码知识图谱成为AI Coding标配

从"上下文注入"到"知识检索"的范式转移

方案 Token消耗 准确性 适用场景
全量上下文 极高 一般 小代码库
知识图谱 低 高 大型项目
RAG 中 高 文档检索

代表项目：Understand-Anything、codegraph、Understand-Anything

趋势2：Skills生态从实验走向生产

Matt Pocock的Skills已达10万星，证明Skills可复用性价值

Skills vs Prompt Engineering：
- ✅ Skills：可版本控制、可分享、可测试
- ❌ Prompt：碎片化、难复用、难测试

754个网络安全技能、隐身浏览器兴起

  • 企业级Agent需要安全审计能力
  • 数据隐私驱动本地化部署需求
