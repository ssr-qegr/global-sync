1. DietrichGebert/ponytail

给你的 AI 编码 Agent 装一个"少说话、多删代码"的老工程师灵魂

TOP 1
⭐ 69,342 Stars🍴 3,576 Forks💻 JavaScript🪪 MIT🤖 Works with 16 agents

项目详细介绍：Ponytail 的定位非常特别——它不是又一个 Agent 框架，也不是又一个 prompt 优化器，而是一份"随插即用"的 Agent 技能包（skill），塞进 Claude Code、Cursor、Codex 等常见 AI 编码 Agent 里，让它们在写代码时先像一位"扎着马尾的老资深"那样思考："这段真的需要写吗？浏览器/框架/标准库是不是本来就有？"作者用一个真实的 FastAPI + React 开源仓库和 12 个功能任务做了 agentic 基准测试，报告显示相比裸 Agent 平均少写约 54% 的代码（最多 94%），成本降低约 20%、速度提升约 27%，且不牺牲任何安全护栏。它把"最好的代码是你没写出来的那行"这句老工程师箴言，做成了可以复现、可被 Agent 真正调用的方法论。

技术亮点

  • Agent-agnostic 设计：不绑定单一模型或 IDE，官方声明兼容 16 种常见 AI 编码 Agent，可以像"skill 插件"一样即插即用。
  • 用真实仓库做基准：在真实的 tiangolo full-stack-fastapi-template 上跑 12 个真实工单，n=4，用 git diff 来评分，比"单轮 prompt 演示"更接近工程落地。
  • 省代码不牺牲安全：相较于粗暴的"写一行就完事"提示词，Ponytail 保留完整的安全与边界检查，避免为了短而漏检。
  • 示范"AI 工程审美"：把"优先用平台原生能力、避免过度依赖、拒绝重复造轮子"沉淀成 Agent 可执行的规则，是一份非常好的教学材料。

使用场景

  • 日常 AI 编码提效：接入 Claude Code / Cursor / Codex 后，让 Agent 在生成代码前先"减法思考"，减少无谓的 wrapper 与依赖。
  • 团队工程规范落地：把"能用原生就不装库、能写一行就不写十行"的规约变成可加载的 Agent skill，而不是只写在 wiki 里。
  • Agent skill 学习范本：想理解"什么是好的 Agent skill 设计"，Ponytail 是一份质量极高的开源参考实现。
  • Code Review 辅助：作为反过度设计的第二双眼睛，帮 reviewer 提前发现"为什么要装 flatpickr 才能选个日期"这类问题。

为什么今天推荐它

  • 它把很多资深工程师的"直觉"显式化了，值得每一个正在依赖 AI 编码的人认真读一遍。
  • 基准测试用真仓库真任务而不是 toy demo，可信度明显高于一般 Agent 项目。
  • MIT 许可 + 与主流 Agent 兼容，落地门槛极低，明天就能接入你手上的 Agent。

2. cpaczek/skylight

用一台 RTL-SDR 把头顶飞过的飞机、卫星和星空实时投影到你家天花板

TOP 2
⭐ 2,970 Stars🍴 355 Forks💻 TypeScript🪪 MIT📡 SDR × 天文 × 实时渲染

项目详细介绍：Skylight 是那种"看一眼就想立刻做一台"的开源硬核创客项目。它用一台便宜的 RTL-SDR 接收头顶飞机广播的 ADS-B 信号，把飞机的机型、航司、目的地和飞行轨迹实时渲染出来，然后通过一台朝上的投影仪投在你家天花板上——真的飞机飞过屋顶的同一时刻，天花板上也有一架同步滑过。背景是纯黑色，投影仪的矩形完全"隐形"，只剩飞机和星空发亮。除了飞机本身，Skylight 还叠加了真实天象：太阳、月亮及月相、亮星与星座连线、肉眼可见的行星，以及包括 ISS 在内的卫星轨道，全部按你的位置和当前时间精确计算。参考实现以旧金山 SFO 为中心，也可以通过 ICAO/IATA 代码切换到全球任意机场。作为加分项，它还支持一台 PTZ 摄像机（VISCA-over-IP + RTSP）自动跟拍它正在投影的那架飞机，配一个 TV 仪表盘和完整的调试 UI。

技术亮点

  • SDR + 实时渲染：本地解码 ADS-B，把 ~1 Hz 的定位插值到 60 fps 平滑运动，同时叠加高度着色、彗尾轨迹和方位罗盘。
  • 真实天象计算：基于 TLE 计算卫星和 ISS 位置，日月星象按经纬度和时间实时定位，还支持前后拖动时间轴。
  • 跨端可玩性强：手机 LAN 控制面板可以调旋转、主题、调色板、过滤器、天空图层等所有参数，配置断电保存。
  • 硬件视觉一体化：PTZ 摄像机 + 视觉追踪能自动锁定并拍摄它正在投影的飞机，用置信度门控做变焦跟拍，工程完整度惊人。

使用场景

  • 周末创客项目：组一套投影仪 + RTL-SDR + 树莓派/迷你主机，就能拥有一片"能飞飞机的天花板星空"。
  • 技术综合练习：一次同时练到 SDR 无线电、TypeScript 实时渲染、天文计算与 PTZ 视觉追踪，学习性价比极高。
  • 科普与教学演示：非常适合家庭、学校或线下活动，把"看不见的电波"以肉眼可见的方式呈现出来。
  • 私人装置艺术：纯黑背景 + 极简发光图形的美学风格，本身就是一件可以放在客厅或办公室的沉浸式装置作品。

为什么今天推荐它

  • 它是"技术极客审美"的完美样本：真实数据、真实物理、真实位置，浪漫但不虚。
  • 作为一个 TypeScript 项目，代码可读性远高于传统 C++ SDR 项目，适合前端 / 全栈同学入门 SDR。
  • 项目结构非常完整（radar、TV 仪表盘、debug UI 都齐全），是学习"如何把一个业余项目做成产品"的极佳案例。
