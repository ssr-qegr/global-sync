AI 视频生产 · Agent Workflow

OpenMontage：让 AI 编程助手成为视频制作工作室

calesthio/OpenMontage

OpenMontage 是一个开源的 agentic video production system，目标不是再造一个传统视频生成器，而是把 Claude Code、Cursor、Copilot、Codex、Windsurf 等 AI 编程助手变成视频制作流程的“总导演”。项目以 pipeline manifest、stage director skill、Python 工具、reviewer skill 和 JSON checkpoint 串起从任务拆解、素材处理到质量自检的端到端流程。

技术亮点
  • Agent-first 架构：没有传统中心化调度器，AI 编程助手本身就是流程编排者。
  • Pipeline + Skill 组合：用 YAML 描述阶段、工具、验收标准，用 Markdown 描述每个阶段的执行方法。
  • 可恢复执行：通过 JSON checkpoint 保存状态，适合长链路视频任务中断后继续推进。
  • 质量门禁：让 Agent 根据 reviewer skill 做 schema 校验、playbook 合规检查和质量复盘。
使用场景

适合技术团队做产品 Demo、教程视频、架构讲解、开源项目宣传片，也适合个人创作者把“脚本—素材—旁白—剪辑—字幕”的复杂链路拆成可审计、可复用的 Agent 工作流。对工程师而言，它的价值不只是做视频，更在于学习如何把复杂创意任务工程化。

Web Agent · 前端智能化

PageAgent.js：把网页变成自然语言可控的 AI 原生应用

alibaba/page-agent

PageAgent.js 是 Alibaba 开源的 JavaScript in-page GUI agent。它的核心思路是把 Agent 直接放进网页里，让用户用自然语言控制界面，而不是依赖脆弱的 XPath、选择器脚本或外部自动化服务。官方定位覆盖 SaaS AI Copilot、智能表单填写、无障碍访问，以及通过浏览器扩展支持多页面控制。

技术亮点
  • 纯前端接入：更像在现有 Web App 中嵌入一个 AI 操作层，而不是另起一套后端自动化系统。
  • 面向业务界面：强调表单填写、后台系统、ERP/CRM 等多步骤点击流程的自然语言化。
  • 可访问性增强：语音命令、屏幕阅读器与自然语言交互能降低复杂 Web 系统使用门槛。
  • 可扩展到多页面：基础能力不依赖浏览器扩展，扩展只作为跨标签页、多页面控制的增强项。
使用场景

适合给内部运营后台、SaaS 产品、管理系统加“AI Copilot”入口，让用户说一句“帮我把这批客户按华北区筛出来并导出”，系统自动完成多步操作。对前端工程师来说，它是理解 GUI Agent、产品内 Copilot 和自然语言交互设计的好样本。

今日观察

这两个项目都不是单点模型能力展示，而是把 AI 放进真实工作流：OpenMontage 处理长链路内容生产，PageAgent.js 处理真实 Web 产品交互。对技术成长更有启发的是，它们都在尝试把“自然语言意图”转成“可执行、可校验、可恢复”的工程流程。
