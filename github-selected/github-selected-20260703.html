1. openclaw/openclaw

运行在你自己设备上、贯穿 20+ 聊天平台的开源个人 AI 助理，"The lobster way"

TOP 1
⭐ 381,521 Stars🍴 79,970 Forks💻 TypeScript🪪 MIT-like🦞 Personal AI Assistant

项目详细介绍：OpenClaw 是一款完全跑在你自己设备上的开源个人 AI 助理。它的产品哲学非常明确：助理才是产品，Gateway 只是控制面——也就是说它并不追求做一个大而全的"Agent 平台"，而是把"一个真正属于你的 AI"塞进你已经在用的所有聊天工具里。目前它已经原生接入了包括 WhatsApp、Telegram、Slack、Discord、Google Chat、Signal、iMessage、IRC、Microsoft Teams、Matrix、飞书、LINE、Mattermost、Nextcloud Talk、Nostr、微信、QQ、WebChat 等 20+ 个渠道；在 macOS/iOS/Android 上支持语音收发，并且提供一个可以由 AI 直接控制的 Canvas 交互层。安装方式非常极简：Node 24 环境下 npx openclaw onboard（或 pnpm/bun），CLI 会一步步引导你完成 gateway、workspace、渠道和 skills 的配置；Windows 用户还能直接装原生 Windows Hub 桌面应用，包含托盘状态、聊天、node 模式与本地 MCP 模式。它由 OpenAI、GitHub、NVIDIA、Vercel 等厂商共同赞助，是过去一年里 GitHub 上增长最快的 AI 项目之一，Stars 已经飙到 38 万+。

技术亮点

  • 单用户 · 本地优先：Gateway 跑在你自己的机器/服务器上，模型 API Key 由你掌握，不上传数据到闭源云端。
  • 多渠道统一网关：用同一份 skill 与记忆体系覆盖 20+ 聊天平台，聊到哪里都是同一个"人"。
  • 语音 + Canvas 多模态：macOS/iOS/Android 上支持语音输入输出，并可渲染由 AI 控制的 Live Canvas，超越纯文本对话。
  • MCP 原生集成：Windows Hub 内置本地 MCP 模式，可以把本机工具、文件系统能力直接暴露给助理调用。

使用场景

  • 私人生产力助理：在飞书、Slack、Telegram 里同时使用同一个助理，安排日程、总结会议、写周报。
  • 家庭/个人知识管家：把家庭 IM（微信/iMessage/WhatsApp）接进来，随时问家庭清单、收纳位置、账单。
  • 开发者工作流：结合 MCP，把 Git、Docker、shell 命令交给助理，用聊天驱动本机操作。
  • 无障碍与无键盘工作流：手机端语音直接指挥，减少切换 App 的成本。

为什么今天推荐它

  • 它把"个人 AI 助理"这个概念从"某某 App 里的一个聊天框"真正做成了跨渠道、跨设备、由你自己拥有的实体。
  • MIT 类协议 + 一句命令上手，非常适合作为周末动手项目搭一个属于自己的 Jarvis。
  • Star 数据+赞助商结构说明它已经度过了"新玩具"阶段，社区和生态都值得押注。

2. bytedance/deer-flow

字节跳动开源的长时程 SuperAgent 框架 2.0，可跑数分钟到数小时的复杂任务

TOP 2
⭐ 75,955 Stars🍴 10,268 Forks💻 Python🪪 MIT🦌 SuperAgent Harness

项目详细介绍：DeerFlow（Deep Exploration and Efficient Research Flow）是字节跳动开源的一套 SuperAgent Harness，2026-02-28 v2.0 发布当天就登顶 GitHub Trending。与"给 LLM 套一层聊天"的常见 Agent 项目不同，DeerFlow 2.0 是从零重写的框架，它把长时程 Agent 需要的所有基础设施打包成一整套：子 Agent 编排、长期记忆、沙盒 & 文件系统、可扩展 Skills、消息 Gateway、上下文工程。它内置了 Claude Code 集成、Session Goals（把"任务目标"当作一等公民管理）、Sub-Agents（父 Agent 派发子任务）、Sandbox（安全执行代码/命令）、长期记忆等能力，能真正处理"跑几十分钟到几小时"的研究、编码和创作任务。部署方式支持 Docker 一键起，也支持本地开发；接入 IM 渠道后可以像 ChatOps 一样在群里驱动 Agent 干活；还内置了 LangSmith / Langfuse Tracing、MCP Server、TUI 终端工作台。字节推荐它搭配 Doubao-Seed-2.0-Code、DeepSeek v3.2、Kimi 2.5 使用，同时集成了 BytePlus 自研的 InfoQuest 智能搜索/爬取工具集。

技术亮点

  • SuperAgent Harness 完整栈：Sub-Agents + Memory + Sandbox + Skills + Gateway，是目前开源生态里最"齐全"的 Agent 底座之一。
  • 长时程任务优先：专为"跑几十分钟到几小时"的研究/编码/创作任务设计，不再是一次性问答。
  • Skills 可扩展：把领域能力抽象成 Skill，Agent 可以按任务拉取，与 Claude Code / Cursor 等外部编码 Agent 协同。
  • 可观测 + 可部署：原生集成 LangSmith / Langfuse Tracing、Docker、MCP Server、TUI，从原型到生产链路完整。

使用场景

  • 深度研究报告：输入一个主题，让 SuperAgent 自主检索、比对、写作，产出多章节调研报告。
  • 长任务编码代理：接入 Claude Code / Codex，作为长时程编码代理跑重构、迁移、升级依赖等大型任务。
  • 企业内部 ChatOps：接入 IM 渠道后，把 Agent 变成"群里的项目经理"，长任务后台跑，结果推送回群。
  • Agent 研发实验床：研究 Sub-Agent 编排、长期记忆、上下文工程、Sandbox 安全的最佳工程实践参考。

为什么今天推荐它

  • 2026 年 Agent 生态的关键词是"长时程"，DeerFlow 2.0 是这类项目里工程完成度最高、生态最完整的开源代表。
  • MIT 协议 + Docker 一键部署 + 完整 Tracing，从"看看"到"用起来"到"改起来"路径很短。
  • 对国内团队特别友好：文档中英日法俄五语齐全，还给出了 Doubao / DeepSeek / Kimi 的推荐搭配。
