codebase-memory-mcp 是一个用 C 写的 MCP（Model Context Protocol）服务器，专注于把任意规模的代码库索引成可供 AI 编辑器实时调用的「长期记忆」。它解决的痛点是：Cursor / Claude Code / Codex 这类 AI 编辑器虽然能写代码，但每次都要重新喂上下文；它把代码符号、依赖关系、文件结构持久化下来，对接 MCP 标准后任何支持 MCP 的客户端都能复用同一份「代码记忆」。性能上用 C 实现意味着对大仓也能保持低延迟检索。

技术亮点

  • MCP 原生：遵循 Anthropic 推动的 MCP 协议，编辑器无需做定制集成。
  • 高性能内核：C 实现的索引引擎，支持大仓增量索引与毫秒级查询。
  • 跨编辑器复用：Cursor / Codex / Claude Code / Gemini CLI 共享同一份「代码大脑」。

典型使用场景

  • 多仓微服务：跨多个 git 仓库快速定位「这个接口在哪一层调用过」。
  • 长期项目记忆：切换分支或停工再回来，AI 仍记得你的设计决策。
  • Code Review 辅助：检索相关历史改动作为 Review 上下文。
