二、大厂重磅动态

🔵 微软Build 2026：Agentic Web全面铺开

据《Microsoft Build 2026 and the Agentic Web》（windowsnews.ai）报道，微软在6月2-3日的Build大会上发布了完整的Agentic Web战略：

GitHub Copilot App：独立桌面应用，从编辑器助手升级为多Agent编排中心。每个Agent在独立git worktree中运行，支持Interactive/Plan/Autopilot三种模式。

Copilot SDK：正式GA，覆盖6种语言（Node.js/TypeScript、Python、Go、.NET、Rust、Java），支持将规划、工具调用、MCP服务器、流式输出嵌入自有应用。

Agent Mesh：点对点覆盖网络，让Agent之间发现和协作，无需中心编排器。

Agent Store：企业级Agent市场，微软抽成15%，类似VS Code扩展模式。

Windows Agent Runtime：系统级服务，在Copilot+ PC的NPU上本地运行轻量Agent，零延迟、零数据外传。

MCP协议标准化：微软已将MCP提交IETF，并在Azure、Windows、Edge、Microsoft 365中原生集成。

⚡ 一句话：微软正在把"Agent"变成操作系统的原生能力，而不是一个独立应用。

🟢 Google Gemma 4 QAT：手机端AI的里程碑

据《Gigazine》和《Android Authority》6月8日报道，Google为Gemma 4系列引入了量化感知训练（QAT）技术：

模型 原始内存 QAT版本(Q4_0) 移动优化版
Gemma 4 E2B 11.4GB 2.9GB 1.1GB
Gemma 4 E2B (纯文本) - - 0.84GB
Gemma 4 12B ~28GB ~6.7GB -
Gemma 4 31B ~72GB ~17GB -

核心突破：
- QAT在训练阶段模拟量化，比训练后量化（PTQ）质量损失更小
- E2B纯文本版仅需0.84GB内存，可在手机端流畅运行
- 支持llama.cpp、Ollama、LM Studio等主流推理框架
- Apache 2.0协议，完全开源可商用

⚡ 一句话：0.84GB跑AI大模型，意味着"每部手机都是AI终端"不再是空话。

🔴 字节Bernini：AI视频编辑的"可控革命"

据《AITOP100》和《ai-damn.com》6月5日报道，字节跳动商业化技术团队开源了视频生成与编辑统一框架Bernini：

架构设计：MLLM语义规划器（导演）+ DiT渲染器（后期），两阶段解耦。

核心能力：
- 主体替换、背景替换、材质替换——不需要手动遮罩
- 参考图/视频引导编辑——支持多元素组合、多角度参考
- 图像/视频植入——海报或视频贴入屏幕/招牌，跟随镜头移动
- SA-3D RoPE技术防止多参考素材"认混"

当前状态：Bernini-R推理代码和权重已开源（Apache 2.0），完整MLLM Planner版本仍在整理中。默认480p/16fps，推荐H100/H800/H200 GPU。

⚡ 一句话：Bernini解决的不是"能不能生成"，而是"能不能按人的想法稳定地生成"。

🟡 MateClaw v1.5.0：国产Agent框架的工程化升级

据《AIBase》6月8日报道，开源AI Agent Runtime框架MateClaw发布v1.5.0：

三大核心升级：
1. Goal Checklist机制：将宏观目标拆解为多个独立验收标准，内置评估器逐条验证并记录证据
2. 自维护知识引擎（LLM Wiki）：引入Wiki-link、知识分层（事实层+经验层）、PageType权限管控、Pipeline事件处理
3. 多用户记忆隔离：通过owner_key和三种可见性scope，确保同一Agent服务多用户时不串记忆

工程细节：新增Claude Opus4.8模型支持、MCP工具超时放宽至60秒、微信/企微/飞书渠道统一媒体管线、指数退避重试提升文件交互可靠性。

⚡ 一句话：v1.4.0解决的是"Agent能围绕目标持续工作"，v1.5.0解决的是"这种持续工作如何被验收、被治理、被多人安全使用"。

三、Agent框架生态更新

OpenSquilla — MetaSkill机制入局

据《铠盒智能》6月5日报道，OpenSquilla以"MetaSkill"（元技能）概念强势入局，与OpenClaw、Hermes形成三足鼎立：

框架 核心优势 代表场景
OpenClaw 5700+预制Skill、最成熟生态 内容自动化、客服FAQ、社交媒体管理
Hermes 深度推理、安全对齐最强 法律分析、金融建模、战略咨询
OpenSquilla MetaSkill动态组合、50+外部工具 ERP对接、CRM自动化、跨平台数据整合

MetaSkill是什么：一个可组合的技能单元，包含触发条件、执行逻辑和输出格式。多个MetaSkill可以动态组合形成复合Agent能力，类似乐高积木。

AgentDoG — 上海AI实验室的Agent安全框架

据《Coze》6月8日报道，上海AI实验室开源AgentDoG框架，专注于AI智能体安全诊断。现有安全工具只能给出"安全/不安全"二元判断，AgentDoG能理解Agent动态交互产生的复杂风险，并解释风险根源。

PaperBanana — Google Research的学术图表生成

据《mer.vin》6月7日报道，Google Research的PaperBanana（论文代号）开源了社区版（MIT协议）。它用5个Agent（Retriever→Planner→Stylist→Visualizer→Critic）将方法论文本转为出版级图表和统计图，支持CLI、Python API、Gradio Studio和MCP Server。

五、行业趋势洞察

趋势1：Skill正在成为Agent时代的"npm"

据《6月GitHub十大爆款开源项目》（m.toutiao.com）分析，2026年开发者开始把经验封装成标准化Skill，Skill被视为Agent世界里的"软件包"。社区讨论Skill是否会成为Agent时代的npm——一个Skill就是一种能力，能力可以被安装、Fork、升级和共享。

趋势2：参数竞赛已死，Agent生态为王

据《6月AI大战》（m.toutiao.com）分析，2026年6月没有哪家公司炫耀参数量。竞争主战场从"谁的模型更强"转向"谁的生态更能让Agent跑起来"。三个关键信号：
1. 工具调用成为新的核心基准（ToolBench > MMLU）
2. 推理效率成为差异化竞争力（DeepSeek V4 Pro降价75%）
3. 开源生态形成正反馈（Llama 4、Qwen3社区贡献爆发）

趋势3：MoE架构全面普及

几乎所有主流大模型都在向MoE迁移。主流MoE模型的激活参数比普遍在15%-25%之间，同样效果下推理成本可降低4-7倍。DeepSeek V4 Pro是典型代表：235B总参数，推理时只激活约47B参数。

趋势4：供应链安全成为焦点

据《The Hacker News》6月6日报道，Miasma蠕虫攻击了73个微软GitHub仓库，Claude Code GitHub Action存在漏洞可让恶意Issue劫持仓库，npm生态遭受IronWorm和Miasma变体攻击。开发者需警惕Agent工具链的供应链风险。
