XiaomiMiMo/MiMo-Code

MiMo-Code 是小米 MiMo 团队开源的终端原生 AI 编程助手，定位是“模型与 Agent 协同进化”的开发工具。它把代码理解、命令执行、Git 操作、长期项目记忆和多模型接入整合到命令行工作流中，适合想研究下一代 Coding Agent 形态的开发者。

技术亮点： 支持一行命令安装和 npm 安装，首次启动可引导配置；内置 MiMo Auto 通道，同时支持连接主流 LLM Provider；具备读写代码、运行命令、管理 Git、持久化记忆等 Agent 能力，并提供主 Agent 切换机制。

使用场景： 大型仓库代码理解、长链路需求拆解、端到端功能开发、CLI 内结对编程，以及对 Coding Agent 架构、记忆系统和工具权限模型进行学习研究。

perplexityai/bumblebee

Bumblebee 是 Perplexity 开源的只读开发者终端扫描器，用来检查本地包、编辑器扩展、浏览器扩展和 AI 工具配置是否暴露在已知软件供应链攻击活动中。它关注的是“开发者机器”这个经常被忽略的供应链入口，而不是只扫描生产环境依赖。

技术亮点： 只读扫描设计降低误操作风险；仓库中的 threat_intel 目录维护公开威胁情报目录；工具面向 macOS 和 Linux 开发机，覆盖 npm、PyPI、Go、RubyGems、Composer、Homebrew、浏览器扩展和 VS Code 风格扩展等常见开发生态。

使用场景： 团队收到供应链投毒告警后快速排查开发机暴露面、在安全响应中定位高风险扩展或包、为研发终端建立轻量级安全自检流程，也适合 Go 开发者学习安全工具的工程化实现。
