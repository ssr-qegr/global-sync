1. usestrix/strix

开源自主 AI 渗透测试工具，让 AI Agent 像真正的黑客一样帮你发现并修复漏洞

TOP 1
⭐ 30,021 Stars🍴 3,242 Forks💻 Python🪪 Apache-2.0🛡️ AI Pentest Agent

项目详细介绍：Strix 不是又一个"给 LLM 套壳的漏洞扫描器"，而是一支能自主运行的 AI 渗透测试 Agent 团队。它把专业渗透测试工程师日常用的整套工具链——HTTP 拦截代理、浏览器自动化、Shell 执行、Python 沙盒、子域名与信息收集等——全部开放给 AI Agent 调用，让 Agent 真正"跑起你的应用"、发现漏洞并写出可复现的 PoC，而不是像传统 SAST 那样吐一堆假阳性。它同时支持从命令行本地运行、通过 Docker 沙箱一键 strix --target ./app-directory，也能作为 GitHub Actions 集成到 CI/CD 流水线中，在合入代码前自动阻断有真实可利用漏洞的 PR，并给出 AI 生成的修复补丁与合规级渗透测试报告。它对接了 OpenAI、Anthropic、Google 等主流 LLM 提供商，是目前"AI Agent 落地做安全"这一方向上工程完成度最高的开源实现之一。

技术亮点

  • 多 Agent 协作架构：多个 AI 渗透测试 Agent 并行分工——侦察、利用、验证——像真实红队一样协同工作，可扩展到大规模应用测试。
  • 真实 PoC 而非假阳性：每个报出的漏洞都要求 Agent 通过 Python 沙盒写出可运行的 exploit 才算数，从源头压掉传统扫描器最痛的假阳性。
  • 完整攻击工具箱：内置 Caido HTTP 代理、无头浏览器（专测 XSS/CSRF/鉴权绕过）、交互式 Shell、OSINT 与指纹识别，覆盖真实红队常用工具集。
  • DevSecOps 原生集成：提供开发者友好的 CLI 与 GitHub Actions，能把安全测试直接前置到 PR 阶段，Auto-fix 生成可合并的补丁 PR。

使用场景

  • 应用安全测试：研发团队在没有专职安全同学的情况下，用 CLI 一键跑一次自主渗透测试，快速定位真正可利用的关键漏洞。
  • 快速渗透测试与合规：把原本几周的第三方 pentest 压缩到几个小时内产出，同时输出合规级报告，方便对接 SOC 2、ISO 27001 等审计。
  • Bug Bounty 自动化：白帽黑客把 Strix 用作"外挂"，加速资产测绘与漏洞验证，直接产出可提交的 PoC。
  • CI/CD 阻断线：作为 GitHub Actions 步骤，让每一个 PR 在合入 main 分支前先跑一轮 Agent 渗透，把安全左移到编码期。

为什么今天推荐它

  • 它是"AI Agent 到底能不能干真活"这个问题的一份漂亮答卷：不是聊天，而是自主执行完整攻击链。
  • Apache-2.0 完全开源、Docker 一键起，本地就能跑，不必把代码交给任何 SaaS。
  • 安全 + AI 是明年最热的技术交汇点之一，早读早占位。

2. altic-dev/FluidVoice

macOS 上最快、也是唯一支持完全本地 STT + AI 增强的开源语音输入应用

TOP 2
⭐ 5,583 Stars🍴 333 Forks💻 Swift🪪 GPL-3.0🎙️ 本地 AI 语音

项目详细介绍：FluidVoice 是一款开源的 macOS 语音输入（Dictation）应用，把"想到即写到"这件事做到了几乎无延迟：说完一句话，光标处的文字几乎同步出现。它同时接入了业界几乎所有主流的语音识别模型，包括 NVIDIA 的 Parakeet Flash / v3 / v2、Nemotron Speech 3.5、Cohere、Apple 原生 Speech 与 Whisper，并支持 Fluid Intelligence——一个完全在本地运行的 AI 增强层，能自动做智能格式化、上下文感知的大小写、标点、数字与符号规范化，让转写出来的文本直接可用于邮件、周报、Prompt。相比 Wispr Flow 这类闭源云端方案，FluidVoice 主打两个字：本地——语音不离开你的 Mac，不需要 API Key、不需要联网，隐私零泄漏。它还提供 Command Mode（用语音在 Mac 上执行动作）与 Write Mode（在任意 App 的输入框里写作或改写文本），并计划扩展到 iOS、Windows 与 Linux。安装方式极简：一行 brew install --cask fluidvoice 即可。

技术亮点

  • 零延迟 Parakeet 实现：1.6 版本重写了 Parakeet 集成，做到"说话的同时文字在跳"，是目前 macOS 端体验最流畅的开源语音输入之一。
  • 模型即插即用：可自由切换 Parakeet、Whisper、Nemotron、Apple Speech 等，兼顾速度、准确度与语言覆盖。
  • Fluid Intelligence 本地增强：纯本地 AI 做后处理与格式化，无 API Key、无云上传，同时提供可选的云端 AI 增强以便进阶用户。
  • 跨 App 通用：Write Mode 可以在任何文本框直接写、改、翻译；Command Mode 可以用语音触发 Mac 上的实际操作。

使用场景

  • 技术写作与文档：写周报、日报、需求评审、技术方案，用语音先出初稿再改，效率提升明显。
  • Prompt 工程：写复杂长 Prompt 时用语音口述再由 Fluid Intelligence 自动整理格式，比键盘更接近自然思维流。
  • 无障碍与手部保护：对手部劳损、腱鞘炎的开发者非常友好，可以在几乎不打字的情况下完成日常工作。
  • 隐私优先场景：公司代码评审、涉密沟通、医疗法律领域，需要保证语音数据不出本机。

为什么今天推荐它

  • 它证明"本地 AI"这条路已经被打通到消费级产品体验：不慢、不糙、不用付订阅。
  • GPL-3.0 全开源，Swift 代码结构清晰，也是学习 macOS 原生应用 + 本地模型集成的极好样本。
  • 作为 Wispr Flow 等闭源云端方案的替代品，个人用户可以彻底把语音数据留在本机。
