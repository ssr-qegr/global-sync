3.2 决策矩阵

  • 需要生产级状态机 → LangGraph
  • 工作流可按角色分解 → CrewAI
  • TypeScript 技术栈 → Mastra
  • 自主编码任务 → OpenHands
  • 最小化抽象 → Smol Agents 或 Agno
  • 自托管平台 → Superagent
  • 微软生态 → AutoGen
